# Проект "Крестики-нолики"
Участники проекта: Бородин Артем, Васильева Снежана, Баранов Дмитрий 
Проект заключает в себе игру под названием "Крестики-нолики". 
Элементы игры: Поле 3х3, символ Х, символ О.
Суть игры: Игроки выбирают свой символ, первым делает шаг игрок, выбравший символ Х. Игрокам нужно выстроить выбранные символы в один ряд или по диагонали. Выигрывает игрок, который смог выстроить 3 символа в ряд или по диагонали. В случае если поле заполнено и ни один из игроков не смог выстроить свои символы в ряд или пор диагонали, объявляется ничья. 
Запуск: Для запуска программы откройте файл text6.sh, для выбора позиции хода сначала введите первое число - номер строки, в которой хотите сделать ход, а после введите второе число - номер столбца, в котором хотите ссделать ход.
